🧹1차."진짜" 캐시 파일 강제 삭제 (CMD)
아까 지운 IconCache.db는 빙산의 일각입니다. 윈도우 10/11은 Microsoft\Windows\Explorer 폴더 안에 크기별(16, 32, 256...) 캐시 파일을 따로 숨겨둡니다. 이건 탐색기가 켜져 있으면 삭제가 안 됩니다.

**명령 프롬프트(CMD)**로 탐색기를 죽이고 수술해야 합니다.

Win 키 -> cmd 검색 -> [관리자 권한으로 실행] 클릭.

아래 명령어들을 한 줄씩 복사해서 붙여넣고 엔터 치세요. (오타 방지)

탐색기 강제 종료:

taskkill /f /im explorer.exe
(바탕화면과 작업표시줄이 사라지고 검은 화면만 남을 겁니다. 정상입니다.)

캐시 폴더로 이동:

cd /d "%localappdata%\Microsoft\Windows\Explorer"
숨겨진 캐시 파일 전멸 (아이콘 & 썸네일):

del iconcache*
del thumbcache*
(혹시 '액세스 거부'가 뜨면 몇 번 더 시도하거나 무시하고 다음으로 넘어가세요.)

탐색기 부활:
start explorer.exe

🧹2차.CMD(명령 프롬프트) 실행

Ctrl + Shift + Esc (작업 관리자)

[새 작업 실행] 클릭

입력창에 cmd 입력 (이번엔 관리자 권한 체크 안 해도 됩니다)
taskkill /f /im dwm.exe
